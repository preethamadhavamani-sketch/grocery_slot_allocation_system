📖 Description

This project is a Grocery Slot Allocation System developed using C++. It assigns customer orders to delivery workers based on a fixed maximum capacity per worker. The system ensures efficient distribution of orders and prevents overloading by automatically assigning delivery slots to workers. It helps in managing delivery operations in an organized and balanced way.
